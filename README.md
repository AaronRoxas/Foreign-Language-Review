# Korean Lecture Review

Interactive, gamified React module for **Foreign Language** (Korean): flashcards, matching, and quiz.

- **Flashcards:** Tap a card to flip between consonant and sound; use Prev/Next to move. ㅇ is explained as “no consonant sound” (placeholder for vowel).
- **Matching:** Click a consonant, then click its sound to form a pair. Matched pairs are highlighted; play again when all 14 are matched.
- **Quiz:** Multiple-choice “What sound does this consonant make?” with instant feedback and score.
